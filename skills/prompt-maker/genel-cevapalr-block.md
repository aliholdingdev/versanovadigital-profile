### D2 — AUTHENTICATION

**D2.0** Auth mekanizması?
```
→ CEVAP: JWT + Session Hybrid + MFA (optional)
  Flow:
    1. Login: username + password (bcrypt verified)
    2. Response: accessToken (JWT, 15 min) + refreshToken (7 days)
    3. Per-request: Authorization: Bearer {accessToken}
    4. Refresh: POST /auth/refresh (sends refreshToken)
    5. Logout: Blacklist token (Redis, 7-day TTL)
  
  MFA (Optional, Premium):
    - TOTP (Google Authenticator)
    - SMS backup codes
    - WebAuthn (FIDO2 — future)
  
  Why hybrid:
    ✅ JWT stateless (API scalability)
    ✅ Session fallback (refresh token provides revocation)
    ✅ Security: Short-lived access, long-lived refresh rotation
```
