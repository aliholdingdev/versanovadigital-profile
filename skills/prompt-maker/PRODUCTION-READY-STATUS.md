## 🚀 HOW TO USE

### Quick Start (5-10 minutes)

**Activate the skill:**
```
"I need a MASTER PROMPT for my backend project"
OR
"Generate a system prompt for my PHP/TypeScript team"
OR
"Create steering rules for my Kiro IDE project"
```

**The skill will:**
1. Ask 4 foundational questions about your project
2. Present 5-question batches (you answer in parallel)
3. Dynamically filter based on your responses
4. Execute web research (50-200+ sources)
5. Score answers using criticality algorithm
6. Generate complete 20-section MASTER PROMPT

**Total interaction:** ~7 minutes for full intake

### For Advanced Users

**Load specific blocks:**
```

Focus on categories: "architecture", "security", "performance"
Complexity level: "advanced"
```

**Customize output:**
